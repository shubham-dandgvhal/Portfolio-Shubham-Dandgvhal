This Portfolio Site

The site you're reading right now — a static, responsive, accessible portfolio built from scratch with HTML, CSS, and vanilla JavaScript. Deployed to GitHub Pages with no build step required.

RoleDesigner & developer
TimelineOne weekend
TypeStatic personal site

The idea

A personal portfolio needs to do three things well: introduce the person, surface their best work, and make it easy to get in touch. I wanted mine to feel modern and fast, with zero build pipeline so it's trivial to update and deploy.

How it's built

  • Plain HTML, CSS, and JavaScript — no frameworks, no bundler.
  • Custom CSS with variables for theming (light & dark) and a mobile-first responsive layout.
  • Interactive features: mobile nav, theme toggle persisted in localStorage, project filter, and client-side contact form validation.
  • Accessible navigation and ARIA labelling; honours prefers-reduced-motion.
  • Deployed via GitHub Pages straight from the main branch.

What I wanted to show

  • That I can design a user-facing experience, not just work in tooling and tickets.
  • That I still enjoy coding hands-on — consultancy and development aren't mutually exclusive.
  • A coherent story: summary, experience timeline, project case studies, credentials, and contact.

What's next
