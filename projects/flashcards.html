Flashcard Study App

A lightweight flashcard tool built to make self-testing less boring. Users flip cards, mark what they got right, and watch their mastery climb across a deck.

RoleSolo developer
TimelineTwo weeks
TypeStudy tool

The idea

I wanted a no-frills way to quiz myself on vocabulary without getting distracted by the dozen features most flashcard apps bundle in. A single deck, a flip, a verdict — that's it.

How it works

  • Cards are defined as a JSON array — front, back, and an optional hint.
  • Clicking a card triggers a CSS 3D flip so the answer is revealed.
  • "Got it" or "Review" buttons push the card to different queues.
  • Progress is stored in localStorage so you can pick up where you left off.

What I learned

  • How CSS perspective and transform-style: preserve-3d work.
  • Using data attributes to keep markup semantic but scriptable.
  • Writing small utility functions instead of one giant handler.

What's next

Adding deck import/export via JSON files, plus a light spaced repetition algorithm so reviews get spaced more intelligently.
